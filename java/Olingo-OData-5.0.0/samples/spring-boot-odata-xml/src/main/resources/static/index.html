üß™ Testing with curl

Example commands to test the service:

# Get specific car

# Get metadata

# Create new car

  -H "Content-Type: application/json" \
  -d '{"Model":"A8","ModelYear":2023,"Price":75000.00,"Currency":"USD"}'
